## 라이브러리 설치
```
pip install pyautogui
pip install pyscreenshot

```

## 실행
vscode, intellij 등으로 코드 실행 후 안내 메시지에 따른다.

## mac
mac의 경우 실행시키면 마우스의 권한을 줘야하는 경우가 생긴다(ex> pyCharm)
보안 및 개인 정보 보호 -> 손쉬운 사용 -> pyCharm을 체크하면 된다.
